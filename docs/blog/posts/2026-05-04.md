# May 4: Redacted Tool Use and a Webinar Worth Bookmarking

Claude started redacting tool use content at some point recently — a useful privacy win. Also: STScI announced their 2026 summer webinar series, and the first one is relevant enough to flag.

## Developer Journal

### Tool use redaction: when did that ship?

Claude Code now redacts tool use content in session logs — it's unclear exactly when this landed, but it's visible now. The specific win: there were Claude memory configurations set up to prevent certain content from being stored, and a redacted tool use path means those memories are genuinely not reaching the storage layer rather than being routed through and then filtered.

The practical implication for this project: sessions that run against AWS, MAST APIs, and production endpoints involve credentials and data that shouldn't end up in memory artifacts. The redaction adds a layer of defense beyond the memory prevention rules.

It's one of those changes that felt important to notice and flag — not obviously breaking, not obviously new, just suddenly better. The "I don't know when they started doing this" reaction is actually a good sign: it shipped quietly and worked without requiring any configuration change.

### STScI 2026 Summer Webinar Series

The Space Telescope Science Institute announced their 2026 summer webinar series. The first webinar isn't until June, but the early ones are described as potentially including unpublished research and methodology details that don't surface in the public documentation.

For JWST Data Analysis specifically: STScI is the source of record for MAST data formats, FITS header conventions, and calibration pipeline outputs. Any session covering FITS structure or data pipeline internals would directly inform some of the edge cases the project has been hitting — the WCS-at-origin issue, the magic-byte validation, the manifest quirks. Worth tracking.

A friend in the channel suggested enrolling, which seems right — free webinars on data pipeline internals from the people who built the pipeline are worth the calendar slot.
