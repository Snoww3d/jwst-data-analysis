# May 2: Opus 4.7, the April Token Tab, and the Tool Shuffle

Hit the Claude limit mid-session, spent time with ChatGPT Codex 5.5, and did some honest accounting on AI tooling costs in April.

<!-- more -->
<!-- enriched -->

## Developer Journal

### Working in Codex 5.5 while the limit resets

Hit the Claude session limit early today, which meant a stint with ChatGPT Codex 5.5 for the task at hand. The task was simple enough that it worked fine — nothing complex, nothing that needed deep project context. Codex handled it.

The broader observation: the limit structure means AI-assisted development increasingly involves context-switching between tools. When it's a simple task, the switch is almost invisible. When it involves nuanced project context (the JWST backend quirks, CLAUDE.md constraints, established patterns), the switch costs context rebuilding. Codex doesn't know about the project's auth model, the processing engine's coordinate space expectations, or the Dependabot configuration conventions.

This has been the ongoing calculus for keeping Gemini as a second option. Today's session reinforced the leaning: Gemini hasn't pulled its weight for code work, and switching to OpenAI as the backup makes more sense given the Codex integration.

### Opus 4.7 is working well here

The online discourse about Opus 4.7 vs 4.6 has been critical — benchmarks, comparisons, the usual. For the specific work pattern on this project (large context, multi-file edits, complex instructions like CLAUDE.md constraints), Opus 4.7 is performing well. The criticism may apply to other use cases; it's not showing up here.

### April token tab: $100 in overage

The April statement came in: $100 in overage tokens burned through the month. That's separate from the base subscription. At the velocity this project runs — daily sessions, long contexts, autonomous runs — the $100/month base rate doesn't cover actual usage. The overage was expected in principle, but seeing it itemized makes the cost of Claude-assisted development concrete.

The trade-off calculation is still clearly in favor: the productivity difference is real and measurable in what shipped in April. But the actual cost per month is closer to $200 than $100.
